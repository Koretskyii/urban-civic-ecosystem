# src

Базова архітектура для NestJS сервера:

- `modules/` — бізнес-модулі (auth, users, cities, etc.)
- `shared/` — спільні компоненти (dto, entities, guards, etc.)
- `config/` — конфігурації (database, app)
- `utils/` — утиліти
- `types/` — типи
- `app/` — основний модуль додатку

Додавайте нові файли та папки за потребою.
